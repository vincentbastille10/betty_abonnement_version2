Code d’intégration

Copiez-collez le code suivant dans votre site (par exemple dans votre page “Contact” ou dans un bandeau) pour afficher votre assistant. Il contient votre identifiant public et transmettra automatiquement votre adresse e‑mail au bot pour recevoir les leads.

<iframe src="{{ base_url }}/chat?public_id={{ public_id }}&embed=1{% if buyer_email %}&buyer_email={{ buyer_email }}{% endif %}" width="100%" height="600" style="border:none; overflow:hidden;"></iframe>

Ajouter le bot via Javascript

Si vous préférez utiliser une balise <script>, utilisez la version ci‑dessous :

<script defer src="{{ base_url }}/static/js/embed.js" data-bot-id="{{ public_id }}"{% if buyer_email %} data-buyer-email="{{ buyer_email }}"{% endif %}></script>

Placez ce script à l’endroit où vous souhaitez afficher le bot.
